### 2024-04-08 代码清理

**已移除的废弃功能：**

| 功能 | 移除原因 |
|------|----------|
| 批量Token健康检查 | 前端未使用，功能冗余 |
| Token统计功能 | 前端未使用，数据可通过列出Token获取 |
| 按模型统计Token数量 | 前端未使用 |
| 流式对话(askStream) | 代码缺陷（调用不存在方法）且前端未使用 |

**保留但简化功能：**
- `updateTokenStatus` / `clearTokens` - 后端API保留，前端未使用但可作为扩展点
- `getTokenDetails` - 后端方法移除，功能合并到列出Token

## 后端API函数实现情况

```
server/routes/
├── /api (对话路由 chat.routes.js)
│   ├── ✔️ POST /ask - 发送消息
│   ├── ✔️ POST /branch - 创建分支
│   ├── ✔️ POST /switch-branch - 切换分支
│   ├── ✔️ GET /conversation-tree - 获取对话树
│   ├── ✔️ GET /node-branches - 获取节点的分支列表
│   └── ✔️ GET /topic-messages - 获取话题消息
│
├── /api/topics (话题路由 topic.routes.js)
│   ├── ✔️ GET / - 获取话题列表
│   ├── ✔️ POST / - 创建话题
│   ├── ✔️ POST /switch - 切换话题
│   ├── ✔️ DELETE / - 删除话题
│   ├── ✔️ GET /current - 获取当前话题
│   └── ✔️ GET /:topicId/messages - 获取话题消息列表
│
├── /api/tokens (Token路由 token.routes.js)
│   ├── ✔️ GET / - 获取Token列表
│   ├── ✔️ POST / - 添加Token
│   ├── ✔️ DELETE / - 删除Token
│   ├── ✔️ PUT /info - 更新Token信息
│   ├── ✔️ PUT /status - 更新Token状态
│   └── ✔️ DELETE /all - 清除所有Token
│
├── /api/models (模型路由 model.routes.js)
│   ├── ✔️ GET / - 获取可用模型列表
│   ├── ✔️ GET /current - 获取当前模型
│   └── ✔️ POST /current - 设置当前模型
│
├── /api/providers (提供商路由 provider.routes.js)
│   ├── ✔️ GET / - 获取所有提供商配置
│   ├── ✔️ POST / - 添加/更新自定义提供商
│   ├── ✔️ DELETE /:name - 删除自定义提供商
│   └── ✔️ POST /test - 测试提供商连接
│
├── /api/ollama (Ollama路由 ollama.routes.js)
│   ├── ⬜ GET /config - 获取Ollama配置
│   ├── ⬜ POST /url - 设置Ollama URL
│   ├── ⬜ POST /enabled - 设置Ollama启用状态
│   ├── ⬜ GET /status - 检测Ollama连接状态
│   ├── ⬜ GET /models - 获取Ollama模型列表
│   ├── ⬜ POST /pull - 拉取Ollama模型
│   └── ⬜ DELETE /models/:model - 删除Ollama模型
│
├── /api/skills (技能路由 skill.routes.js)
│   ├── ⬜ GET / - 获取技能列表(支持搜索)
│   ├── ⬜ POST / - 添加自定义技能
│   └── ⬜ DELETE /:skillId - 删除技能
│
├── /api/theme (主题路由 theme.routes.js)
│   ├── ✔️ GET / - 获取当前主题
│   └── ✔️ POST / - 设置主题
│
└── /api/tree (树形结构路由 tree.routes.js)
    ├── ✔️ GET /:topicId - 获取话题的完整对话树
    ├── ✔️ GET /node/:nodeId - 获取节点详细信息
    ├── ✔️ PUT /node/:nodeId - 编辑节点内容
    ├── ✔️ POST /node/:nodeId/copy - 复制节点
    ├── ✔️ DELETE /node/:nodeId - 删除节点
    ├── ✔️ GET /positions/:topicId - 获取话题的节点位置
    ├── ✔️ POST /positions/:topicId - 保存话题的节点位置
    ├── ✔️ DELETE /positions/:topicId - 重置话题的节点位置
    ├── ✔️ GET /viewport/:topicId - 获取话题的视口位置
    ├── ✔️ POST /viewport/:topicId - 保存话题的视口位置
    └── ✔️ DELETE /viewport/:topicId - 重置话题的视口位置
```
